type, public, abstract :: abstract_vector

Abstract type from which users need to extend in order to define the vectors used by LightKrylov. Upon extension, the user needs to provide a handful of type-bound procedures:

  • zero(self)
  • scal(self, alpha)
  • axpby(self, alpha, vec, beta)
  • dot(self, vec)
