public subroutine rational_arnoldi_factorization(A, X, H, G, sigma, info, solve, linsolve_opts, preconditioner, kstart, kend, verbosity, tol, transpose)

Implements the Rational Arnoldi Factorization for arbitrary square linear operators. Only the single pole case is considered with the exception of the last iteration (k=kend) where the pole is set to +infinity to define a proper Rational Arnoldi factorization.
